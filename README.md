# Dendi-Learn 🌍

> Plateforme web d'apprentissage de la langue Dendi — pour préserver et transmettre une langue du nord du Bénin à l'ère numérique.

## 🎯 Pourquoi ce projet ?

Le Dendi est une langue parlée dans le nord du Bénin, mais elle n'existait sur aucune plateforme d'apprentissage en ligne. Dendi-Learn est né d'une volonté de préservation culturelle et linguistique — pas d'une logique commerciale.

Ce projet est entièrement gratuit et open source.

## ✨ Fonctionnalités

- 📚 Flashcards interactives par catégorie
- 🧠 Mode quiz pour tester ses connaissances
- 👤 Authentification utilisateur
- 📈 Suivi de progression personnalisé
- 🌙 Thème clair / sombre

## 🛠️ Stack technique

| Couche | Technologie |
|--------|-------------|
| Frontend | Next.js + TypeScript |
| Base de données | Supabase (PostgreSQL) |
| Auth | Supabase Auth |
| Déploiement | Vercel |

## 🚀 Lancer le projet

```bash
git clone https://github.com/GhostDRAKEN/dendi-learn
cd dendi-learn
npm install
npm run dev
```

Créer un fichier `.env.local` :
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
